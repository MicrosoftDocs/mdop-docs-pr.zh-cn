# Microsoft BitLocker 管理和监视1管理员指南

Microsoft BitLocker 管理和监视（MBAM）提供简化的管理界面，可用于管理 BitLocker 驱动器加密。 使用 MBAM，你可以选择适合你的企业的 BitLocker 加密策略选项，然后使用它们来监视客户端是否符合这些策略。 您还可以报告单个计算机和整个企业的加密状态。 此外，当用户忘记其 PIN 或密码时，或者当其 BIOS 或启动记录发生更改时，您可以访问恢复密钥信息。
