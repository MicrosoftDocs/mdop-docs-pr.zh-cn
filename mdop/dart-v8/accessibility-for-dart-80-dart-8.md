# DaRT 8.0 的辅助功能


Microsoft 致力于使其产品和服务更易于每个人使用。 本部分提供有关使残障人士更易于访问本产品及其相应文档的功能和服务的信息。

## 通过几个键击访问任何命令


访问键允许你通过按几个键来快速使用命令。 您可以使用两个键击访问大多数命令。 若要使用访问键，请执行以下操作：

1.  按 ALT。

    键盘快捷方式将显示在当前视图中可用的每个功能的上方。

2.  按下要使用的功能上的键盘快捷方式中所显示的字母。

**注意** 若要取消正在执行的操作并隐藏键盘快捷方式，请按 ALT。
