# 如何使用 DaRT 恢复映像向导来创建恢复映像


Microsoft 诊断和恢复工具集（DaRT）7包含在 Windows 中用于创建可启动的国际标准化组织（ISO）映像的**DaRT 恢复映像向导**。 ISO 图像是表示 CD 的原始内容的文件。

**DaRT 恢复映像向导**需要以下信息：

-   **启动映像**-必须提供创建 DaRT 恢复映像所需的 WINDOWS 7 DVD 或 windows 7 源文件的路径。

-   **工具选择**"您可以选择要包含在 DaRT 恢复映像中的工具。

-   **远程连接**您可以选择是否希望 DaRT 恢复映像包括在支持人员和最终用户计算机之间建立远程连接的能力。

-   **适用于 windows 的调试工具**-系统要求你提供 Windows 调试工具的位置。

-   **独立系统 Sweeper 的定义**您可以决定是否在创建恢复映像或稍后下载定义时下载最新的定义。

-   **驱动程序**？系统会询问你是否要将驱动程序添加到 ISO 映像。

-   **其他文件**？您可以将文件添加到可以帮助诊断问题的 ISO 映像。

-   **Iso 镜像位置**？系统会要求你指定 ISO 映像所在的位置。

-   **Cd/DVD 驱动器**？系统将要求您指定是否应使用 CD 或 dvd 驱动器刻录 CD 或 dvd。

**注意** ISO 映像大小可能会有所不同，具体取决于在**DaRT 恢复映像向导**中选择的工具。

 

## 使用 DaRT 恢复映像向导创建恢复映像


按照以下说明使用**Dart 恢复映像向导**创建 dart 恢复映像。

### 选择要包含在 DaRT 恢复映像上的工具

**DaRT 恢复映像向导**显示一个**工具选择**对话框。 通过突出显示工具，然后单击 "**启用**" 或 "**禁用**" 按钮，可以从要包括在 DaRT 恢复映像中的工具列表中选择或删除工具。

选择要包括在恢复映像中的所有工具后，单击 "**下一步**"。

### 添加允许远程连接的选项

你可以选择 "**允许远程连接**" 复选框以在 "**诊断和恢复工具集**" 窗口中提供选项，以便在帮助程序代理和最终用户计算机之间建立远程连接。 在帮助台代理建立远程连接后，他们可以从远程位置从最终用户计算机上运行 DaRT 工具。

你可以选中 "**指定端口号"** 复选框以输入建立远程连接时将使用的特定端口号。 你可以指定一个介于1和65535之间的端口号。 建议端口号为1024或更高，以将冲突可能性降至最低。

你还可以创建最终用户建立远程连接时将收到的自定义消息。 该消息最多可以为2048个字符。

有关远程运行 DaRT 工具的详细信息，请参阅[如何使用 DaRT 恢复映像恢复远程计算机](how-to-recover-remote-computers-using-the-dart-recovery-image-dart-7.md)。

### 将 Windows 调试工具添加到 DaRT 恢复映像

在**DaRT 恢复映像向导**的 "**崩溃分析器**" 对话框中，系统会要求你指定 Windows 调试工具的位置。 如果您没有这些工具的副本，则可以从 Microsoft 下载它们。 向导中提供了指向下载页面的以下链接：[下载并安装适用于 Windows 的调试工具](https://go.microsoft.com/fwlink/?LinkId=99934)。

你可以在运行**DaRT 恢复映像向导**的计算机上指定调试工具的位置，也可以决定使用位于目标计算机上的工具。 如果你决定在另一台计算机上使用副本，则必须确保在你诊断崩溃的每台计算机上安装工具。

**注意** 如果你在 ISO 映像中包含**崩溃分析器**，我们建议你还包括适用于 Windows 的调试工具。

 

请按照以下步骤添加适用于 Windows 的调试工具：

1.  可选单击超链接以下载适用于 Windows 的调试工具。

2.  选择下列选项之一：

    -   **在以下位置使用 Windows 调试工具**。 如果选择此选项，则可以浏览到工具的位置。

    -   **在要修复的系统上找到用于 Windows 的调试工具**。 如果选择此选项，在问题计算机上找不到 Windows 调试工具时，**崩溃分析器**将无法正常工作。

3.  完成后，单击 "**下一步**"。

### 将独立系统 Sweeper 的定义添加到 DaRT 恢复映像

定义是已知恶意软件和其他潜在的垃圾软件的存储库。 由于正在不断开发恶意软件，因此**独立系统 Sweeper**依赖于当前定义来确定尝试在计算机上安装、运行或更改设置的软件是潜在的有害软件还是恶意软件。

若要包括 DaRT 恢复映像中的最新定义（推荐），请单击 **"是，下载最新的定义。** 定义更新将自动启动。 必须连接到 Internet 才能完成此过程。

若要跳过定义更新，请单击 "**否，稍后手动下载定义"**。 DaRT 恢复映像中将不包含定义。

如果你决定不在恢复映像上包含最新的定义，或者恢复映像中包含的定义在你准备好使用**独立系统 Sweeper**时不再是最新的，请按照**独立系统 Sweeper**中提供的说明，在开始扫描之前获取最新的定义。

**重要提示** 如果没有定义，则无法扫描。

 

完成后，单击 "**下一步**"。

### 将驱动程序添加到 DaRT 恢复映像

**小心** 默认情况下，当你将驱动程序添加到 DaRT 恢复映像时，位于该文件夹中的所有其他文件和子文件夹将添加到恢复映像中。 有关详细信息，请参阅[DaRT 7.0 的疑难解答](troubleshooting-dart-70-new-ia.md)。

 

你应该在恢复映像上包括修复计算机时可能需要的 DaRT7 的其他驱动程序。 这些通常包括 Windows DVD 中未包含的存储或网络控制器。

**重要提示** 选择要包括的驱动程序时，请注意，DaRT 中不支持无线连接（如蓝牙或 802.11 a/b/n）。

 

**将存储或网络控制器驱动程序添加到恢复映像**

1.  在**DaRT 恢复映像向导**的 "**其他驱动程序**" 对话框中，单击 "**添加设备**"。

2.  通过浏览找到要为驱动程序添加的文件，然后单击 "**打开**"。

    **注意** **驱动程序**文件由存储或网络控制器的制造商提供。

     

3.  对要包括的每个驱动程序重复步骤1和2。

4.  完成后，单击 "**下一步**"。

### 将文件添加到 DaRT 恢复映像

按照以下步骤将文件添加到恢复映像，以便你可以使用它们来诊断计算机问题。

1.  在**DaRT 恢复映像向导**的 "**其他文件**" 对话框中，单击 "**显示文件**"。 这将打开一个 "资源管理器" 窗口，显示包含共享文件的文件夹。

2.  在对话框中列出的文件夹中创建一个子文件夹。

3.  将所需的文件复制到新的子文件夹。

4.  完成后，单击 "**下一步"。**

### 选择包含 DaRT 恢复映像的 ISO 的位置

按照以下步骤指定创建 ISO 映像的位置：

1.  在 " **DaRT 恢复映像向导**" 的 "**创建启动映像**" 对话框中，单击 "**浏览**"。

2.  浏览到 "**另存为**" 窗口中的首选位置，然后单击 "**保存**"。

3.  完成后，单击 "**下一步**"。

ISO 映像的大小会有所不同，具体取决于你选择的工具和你在向导中添加的文件。

向导要求 ISO 映像具有 **.iso**文件扩展名，因为刻录 CD 或 DVD 的大多数程序都需要该扩展。 如果不指定其他位置，将在桌面上创建使用名称**DaRT70**的 iso 图像。

### 将恢复映像刻录到 CD 或 DVD

如果**DaRT 恢复映像向导**检测到计算机上兼容的 cd-rw 驱动器，它可以将 ISO 映像刻录到光盘。 如果你想要刻录 CD 或 DVD，并且向导无法识别你的驱动器，则必须使用另一个程序，例如你的驱动器附带的程序。 你可以使用 duplicator、复制服务或 CD 或 DVD 刻录软件来制作任何其他副本。

1.  在 "刻录到**DaRT 恢复映像向导**的**可写入 cd/dvd** " 对话框中，选择 "**将映像刻录到以下可录制的 cd/dvd 驱动器"**。

2.  选择 CD 或 DVD 驱动器。

    **注意** 如果无法识别驱动器且安装了新驱动器，则可以单击 "**刷新驱动器列表**" 以强制向导更新可用驱动器列表。

     

3.  单击“下一步”****。
