# 管理连接组


连接组使程序包中的应用程序可以在虚拟环境中相互交互，而不是与系统其余部分隔离。 通过使用连接组，管理员可以独立管理程序包，并且可以避免将同一应用程序多次添加到客户端计算机。

**注意** 在以前版本的 App-v 5.0 中，连接组被称为 "动态套件合成"。
