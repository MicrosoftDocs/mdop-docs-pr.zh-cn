# 部署 MED-V 工作区程序包


测试 Microsoft 企业版桌面虚拟化（MED-V）2.0 安装程序包后，您可以使用公司的预配软件和其他应用程序的首选方法在整个企业中部署它。

**注意** 准备好部署时，建议通过运行 MED-V 工作区安装程序包中包含的 setup.exe 可执行程序来安装 MED-V 工作区。

 

**警告** 必须先安装 MED-V 主机代理，然后才能安装 MED-V 工作区。
