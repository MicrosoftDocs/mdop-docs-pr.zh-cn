# MDOP 信息体验


Microsoft 桌面优化包（MDOP）是一套可用作软件保障客户订阅的技术。 MDOP 帮助改进兼容性和管理、减少支持成本、改善资产管理和改进策略控制。

MDOP 信息体验提供了产品文档、视频、博客和其他资源，可帮助用户利用 MDOP 技术实现和优化其体验。 你可以通过在[Facebook](https://go.microsoft.com/fwlink/p/?LinkId=242445)或[Twitter](https://go.microsoft.com/fwlink/p/?LinkId=242447)上关注我们来了解更新和事件。

## <a href="" id="bkmk-getmdop"></a>如何获取 MDOP
