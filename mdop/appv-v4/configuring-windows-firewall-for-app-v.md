# 为 App-V 配置 Windows 防火墙


保护 app-v 基础结构中的组件之间的通信仅是保护环境安全的一个元素。 在服务器上使用防火墙程序可帮助减少受攻击面。 本部分中的主题提供了配置 Windows 内置防火墙功能的过程，这些功能支持 App-v。 这些过程假设你已安装管理服务器，并且已针对 RTSPS 通信进行配置。 备注包含在用于通信的流服务器和环境的过程中。
