# “安装报告”页（了解详细信息）


使用 "**安装报告**" 页面查看排序过程的结果。 这些报表可帮助诊断排序期间可能出现的常见问题。

此页面包含以下元素：

<a href="" id="information-pane"></a>**信息窗格**  
显示有关整个排序过程的问题和其他信息。
