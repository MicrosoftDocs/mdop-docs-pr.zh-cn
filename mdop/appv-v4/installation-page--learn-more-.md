# “安装”页（了解详细信息）


使用 "**安装**" 页面运行完成应用程序安装所需的其他安装文件。

此页面包含以下元素：

<a href="" id="run"></a>**运行**  
打开 "**选择安装文件**" 对话框。 指定其他安装文件、Windows Installer 和可执行程序文件的位置，然后单击 "**打开**"。

<a href="" id="i-am-finished-installing"></a>**我已完成安装**  
启用 "**下一步**" 按钮。 如果没有与此应用程序相关联的其他安装文件，请选择此选项。
