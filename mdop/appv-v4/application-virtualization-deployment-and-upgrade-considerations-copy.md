# Application Virtualization 部署和升级注意事项


开始部署 Microsoft Application Virtualization （App-v）之前，可能需要查看环境要求，包括安装各种应用程序虚拟化组件的硬件和软件要求。 此外，如果您从早期版本升级，本部分中的主题将提供有关如何升级当前 Sequencer、服务器和客户端版本的信息。
