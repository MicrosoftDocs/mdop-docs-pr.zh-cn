# “完成”页


使用 "**完成**" 页面查看程序包完成报告。 排序期间遇到的任何问题都将显示在 "**成功的虚拟应用程序包报告**" 窗格中。 此信息也可在将程序包保存在名为 Report.xml 的文件中的目录中。

此页面包含以下元素：

<a href="" id="close"></a>**关闭**  
关闭 "**创建新程序包" 向导**。
