# 如何编辑 OSD 文件


通过添加或删除元素或属性，使用以下过程修改序列化应用程序包的开放式软件描述符（OSD）文件。

**注意** 某些元素没有属性，因此不可能将属性添加到每个元素。

 

**重要提示** 如果使用 OSD 编辑器更改 sft 文件名，即 OSD 文件中 CODEBASE 元素的 HREF 属性，则必须使用 "**另存为**" 命令将更改保存到项目文件。

 

**添加元素**

1.  单击 " **OSD 文件**" 选项卡。

2.  在导航窗格中，选择要修改的序列化应用程序包的 OSD 文件。

3.  在导航窗格中，右键单击要修改的元素。 在菜单上，选择 "**元素**"，然后选择 "**添加**"。

4.  从菜单中选择要添加的元素，例如**Codebase**。

5.  从 "**文件**" 菜单中，选择 "**保存**"。

**删除元素**

1.  单击 " **OSD 文件**" 选项卡。

2.  在导航窗格中，选择要修改的序列化应用程序包的 OSD 文件。

3.  在导航窗格中，右键单击要删除的元素。 在菜单上，选择 "**元素**"，然后选择 "**删除**"。

4.  从 "**文件**" 菜单中，选择 "**保存**"。

**添加属性**

1.  单击 " **OSD 文件**" 选项卡。

2.  在导航窗格中，选择要修改的序列化应用程序包的 OSD 文件。

3.  在左窗格中，右键单击要向其添加属性的元素。 在菜单上，选择 "**属性**"，然后选择 "**添加**"，从列出的可用属性中选择。

4.  从 "**文件**" 菜单中，选择 "**保存**"。

**删除属性**

1.  单击 " **OSD 文件**" 选项卡。

2.  在导航窗格中，选择要修改的序列化应用程序包的 OSD 文件。

3.  在导航窗格中，右键单击要从中删除属性的元素。 在菜单上，选择 "**属性**"，然后选择 "**删除**"，选择要删除的属性。

4.  从 "**文件**" 菜单中，选择 "**保存**"。
