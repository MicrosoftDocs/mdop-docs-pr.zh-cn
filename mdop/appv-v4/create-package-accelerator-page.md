# “创建包加速器”页


使用 "**创建包加速器**" 页面指定程序包加速器的保存位置。

此页面包含以下元素：

<a href="" id="browse"></a>**浏览**  
单击 "**浏览**" 以指定程序包加速器的保存位置。
