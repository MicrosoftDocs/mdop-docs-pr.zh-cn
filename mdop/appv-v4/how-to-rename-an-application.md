# 如何重命名应用程序


你可以通过应用程序虚拟化服务器管理控制台重命名应用程序。

**重命名应用程序**

1.  在管理控制台的左窗格中，单击 "**应用程序**"。

2.  右键单击该应用程序，然后选择 "**重命名**"。

3.  输入新名称，然后按**enter**，或单击框外的 "取消"。

    **注意** 你还可以在右窗格中突出显示应用程序，然后按**F2**。
