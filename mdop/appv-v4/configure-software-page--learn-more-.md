# “配置软件”页（了解详细信息）


使用 "**配置软件**" 页面运行每个程序以完成安装后的任何配置任务。 例如，此步骤有助于配置任何关联的应用程序许可协议。

此页面包含以下元素：

<a href="" id="run-selected"></a>**运行选定的**  
仅打开与应用程序相关联的选定程序。

<a href="" id="run-all"></a>**全部运行**  
打开与应用程序相关联的所有程序。
