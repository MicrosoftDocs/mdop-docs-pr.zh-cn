# 如何禁用或修改断开连接操作模式设置


使用应用程序虚拟化客户端中的以下过程禁用或修改已断开连接的操作模式设置。

**禁用断开连接的操作**

1.  右键单击控制台中的 "**应用程序虚拟化**" 节点，然后从弹出菜单中选择 "**属性**"。

2.  单击 "**连接**" 选项卡，然后清除 "**允许断开连接的操作**" 复选框。

3.  单击 **"确定"** 接受更改。

**更改超时**

1.  右键单击控制台中的 "**应用程序虚拟化**" 节点，然后从弹出菜单中选择 "**属性**"。

2.  单击 "**连接**" 选项卡，然后选中 "**将断开连接的操作限制为**" 复选框。

3.  在字段中，输入1到999999的值（表示天数）。 默认值为90天。

4.  单击 **"确定"** 接受更改。

**脱机工作**

1.  右键单击控制台中的 "**应用程序虚拟化**" 节点，然后从弹出菜单中选择 "**属性**"。

2.  单击 "**连接**" 选项卡，然后选中 "**脱机工作**" 复选框。

3.  单击 **"确定"** 接受更改。
