# “过大的程序包”对话框 (App-V 4.6 SP1)


未压缩的程序包大小超过了推荐的 4 GB 程序包大小限制。 为了减小程序包的大小，已启用软件包压缩。

在 "**特大程序包**" 对话框中，使用以下过程配置软件包压缩。

1.  在 App-v Sequencer 控制台中，单击 "**部署**" 选项卡。

2.  若要启用程序包压缩，请选中 "**压缩包**" 复选框。 若要禁用程序包压缩，请清除 "**压缩包**" 复选框。

3.  若要保存程序包，请单击 "**文件**  /  **保存**"。
