# 基于电子软件分发的方案


如果您计划对您的 Microsoft Application Virtualization 环境使用电子软件分发（ESD）部署方案，请务必了解进入和受该决策影响的因素。 本部分中的主题介绍 ESD 方案，并提供有关程序包传递方法、传输协议和在部署策略中需要考虑的外部组件的信息。 你还可以使用本节中的过程完成部署，从服务器配置阶段到部署验证阶段。
