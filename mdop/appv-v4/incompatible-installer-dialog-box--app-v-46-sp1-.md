# “安装程序不兼容”对话框 (App-V 4.6 SP1)


应用程序未正确安装。

使用以下过程验证你尝试序列的应用程序是否与运行 sequencer 的计算机兼容。

1.  关闭 App-v 排序器。

2.  在运行 Sequencer 的计算机上安装应用程序，但不监视安装。

3.  验证应用程序是否兼容以及是否可以成功安装。
