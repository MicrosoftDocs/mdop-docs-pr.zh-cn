# Application Virtualization 属性：“权限”选项卡


使用 "**应用程序虚拟化属性**" 对话框的 "**权限**" 选项卡指定哪些活动可用于本地计算机上的非管理员。 选中与要对所有用户可用的项目对应的复选框。 没有管理权限的用户可以查看此页面并查看已选择的项目，但不能修改列表。
