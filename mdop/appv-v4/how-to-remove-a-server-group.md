# 如何删除服务器组


在应用程序虚拟化服务器管理控制台中，你可以使用以下过程从你的应用程序虚拟化系统中删除服务器组（包括组中的所有服务器）。

**删除服务器组**

1.  单击左窗格中的 "**服务器组**" 节点以展开服务器组列表。

2.  右键单击所需的服务器组，然后选择 "**删除**"。

3.  出现提示时，单击 **"是"**。
