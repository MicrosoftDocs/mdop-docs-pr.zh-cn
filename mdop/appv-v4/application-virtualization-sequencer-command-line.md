# Application Virtualization Sequencer 命令行


你可以使用 Microsoft Application Virtualization （App-v） Sequencer 使用命令行创建虚拟应用程序包。 这种排序方法比使用**顺序**向导排序更快，并且在对大量应用程序进行排序时非常理想。
