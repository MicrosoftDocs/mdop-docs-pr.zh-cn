# 关于本指南

-   系统要求和支持的平台

-   环境先决条件和注意事项

-   安装 MED-V 服务器、管理和客户端的分步说明

-   有关配置 MED-V 图像、工作区和策略的分步说明

-   部署 MED-V 工作区的分步说明

-   MED-V 客户端操作
