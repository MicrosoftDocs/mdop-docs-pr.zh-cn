# 如何安装 MED-V 客户端


在用户计算机上安装 MED-V 程序包之前，请确保将图像上载到 Web 或将其推送到 "图像预调试" 文件夹。

若要使用企业软件分发中心安装 MED-V 客户端，请参阅有关如何使用正在使用的系统安装应用程序的文档。
