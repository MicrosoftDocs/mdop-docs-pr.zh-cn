# 回收站命令


"**回收站**" 选项卡：

-   显示已从存档中删除的组策略对象（Gpo）的列表。

-   提供快捷菜单，其中包含用于管理 Gpo 和显示 Gpo 报告的命令。

-   显示具有访问选定 GPO 的权限的组和用户的列表。

右键单击此选项卡上的 "**组策略对象**" 列表将显示快捷菜单，包括以下任何适用选项：
